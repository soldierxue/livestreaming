RTMP/HTTP-FLV/HLS/DASH 开源直播平台（托管在 AWS EC2 + Cloudfront）

这台机器上，我们跑着 SRS（Simple RTMP Server）和 Ngnix + RTMP 模块，目的是演示如何自建一个直播平台。

申明：只为了演示学习目的，直播源支持本地的视频，用户也可以用 OBS 尝试推流到本服务器

直播频道及相关播放地址：

演示本地 ffmpeg 推流，视频源为固定的本地 mp4 视频，点击直接浏览器 HTML5 播放源 mp4 视频：
  • 频道 I：接受播放透过 Nginx + RTMP 支持的直播流，DASH 方式
  • 频道 II：接受播放透过 Nginx + RTMP 支持的直播流，HLS 方式
  • 频道 III：接受播放透过 SRS 支持的直播流，HTTP-FLV 方式
  • 个人频道 ：通过 OBS推流到 SRS，HTTP-FLV 拉流播放
  • SRS Flash 播放器：接受播放透过 SRS 支持的直播流，RTMP 方式 （需要暴露 EC2 实例的公有 IP，还没想好比较好的管控措施，故暂时不提供 RTMP 播放地址
